# City of Bits

Building real-time urban intelligence — the book, the code, and the community.

**City of Bits** is a book about turning cities into living, sensed, agent-native
systems, plus the open companion code you build alongside it.

### Explore

- 📟 **[cityscope](https://github.com/city-of-bits/cityscope)** — companion code from the
  book: build a real-time urban intelligence dashboard, chapter by chapter. *(MIT)*
- 🌐 **[city-of-bits.com](https://city-of-bits.com)** — the book, articles, podcast, and newsletter.
