# City of Bits

Turning cities into living, sensed, agent-native systems — the book, the code you can
actually run, and the argument for why it should be built this way.

Written and maintained by [Vlad](https://city-of-bits.com) — PhD research in Physical AI,
two funded/exited startups, now building the open reference stack in public, chapter by
chapter.

### Explore

- 📟 **[cityscope](https://github.com/city-of-bits/cityscope)** — companion code from the
  book: build a real-time urban intelligence dashboard, one build per chapter, from a single
  sensor to a full city data bus. *(MIT)*
- 📜 **[manifesto](https://github.com/city-of-bits/manifesto)** — the Web 4 Objects
  Manifesto: ten principles for building the City of Bits, open to challenge chunk by chunk.
- 🌐 **[city-of-bits.com](https://city-of-bits.com)** — the book, articles, podcast, and
  newsletter.
